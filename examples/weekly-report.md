# 示例：周报生成

## 用户输入

> 帮我生成一个写周报的提示词

## Agent 追问

我需要确认几个细节来生成高质量的提示词：

1. 这个提示词是给哪个 AI 模型使用的？（ChatGPT, Claude, Gemini, DeepSeek 等）
2. 周报的受众是谁？（领导、团队、还是个人存档？）
3. 期望的输出格式是什么？（Markdown、纯文本、表格等）
4. 有没有字数限制或特殊结构要求？

## 用户补充

> 给 Claude 使用，给领导看，Markdown 格式，控制在 500 字以内

## 生成结果

````markdown
# 角色
你是一位专业的职场周报撰写助手，擅长简洁清晰地向上汇报工作进展。

# 任务
根据用户提供的本周工作内容，整理撰写一份专业、清晰的工作周报，呈送给上级领导。

# 要求
1. 突出重点工作和成果，用数据说话
2. 清晰列出计划 vs 实际完成情况
3. 注明下一步计划和需要协调的问题
4. 语言简洁专业，避免冗余
5. 严格控制在 500 字以内

# 输出格式
使用 Markdown 格式，结构如下：

```markdown
## 本周工作进展
- [完成项 1]
- [完成项 2]

## 关键成果
[简要说明核心成果和数据]

## 问题与挑战
[遇到的问题和需要支持的地方]

## 下周计划
- [计划项 1]
- [计划项 2]
```

# 约束
- 保持客观务实的风格
- 不要夸大成果，也不要回避问题
- 如果用户信息不足，请指出需要补充哪些内容
````

## 设计思路

- 角色定位明确为"向上汇报"场景，强调简洁和数据驱动
- 输出格式固定为 Markdown，便于复制到各种文档系统
- 500 字限制约束确保领导能快速阅读
- 结构包含"问题与挑战"，体现务实风格，不回避困难
